# プロダクト概要

## 家族用 ToDo アプリ - サーバーレス イベントソーシング

イベントソーシングと CQRS アーキテクチャパターンを採用したサーバーレス家族用 ToDo アプリケーションです。家族が協力して ToDo アイテムを管理し、完全な監査証跡とリアルタイム更新を提供します。

### 主要機能

- **家族ベースマルチテナンシー**: 各家族のデータアクセスを分離
- **イベントソーシング**: すべての ToDo 操作の完全な監査証跡
- **CQRS**: 最適なパフォーマンスのための読み取り/書き込みモデル分離
- **Passkey 認証**: Amazon Cognito による WebAuthn ベースのパスワードレスログイン
- **リアルタイム更新**: DynamoDB Streams によるライブデータ同期
- **サーバーレスアーキテクチャ**: 自動スケーリング対応の AWS Lambda 関数

### 主要ビジネスルール

- すべての操作は家族スコープ（マルチテナント分離）
- すべての変更は監査コンプライアンスのための不変イベントを生成
- 楽観的ロックにより同時変更の競合を防止
- 論理削除によりコンプライアンスと復旧のためのデータ保持
- 後方互換性のためのバージョンベースイベントスキーマ進化

### パフォーマンス目標

- API レスポンス時間: <200ms (p95)
- DynamoDB 操作: 書き込み<20ms、読み取り<10ms
- Lambda コールドスタート: <500ms (p99)
- エラー率: <0.1%
- 可用性: 99.9%

### 言語設定

- **チャット応答言語**: 日本語
- **ドキュメント言語**: 日本語
- **コメント言語**: 日本語（コード内コメント）
